## Установка 

- .env.example  copy .env.
- composer install.
- php artisan migrate.

## О программе 

Позволяет пользователям создать короткие линки  и имет свой лист линков. 

## Как использовать ?    

- регистрацыя.
- Будет уже доступен /links урл где будет список линков.
- можно тогда добавить редактировать и удалить линков.
